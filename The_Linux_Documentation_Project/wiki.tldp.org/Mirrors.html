Linux Documentation Project Mirrors

The following sites mirror the LDP pages, please use the site nearest to you. DAILY means mirrored daily, DAILY*N means mirrored N times per day, WEEKLY means mirrored weekly, WEEKLY*N means mirrored N times a week.

Mirror Date

If you are using a mirror site, the date on the main page of the mirror (upper right corner) should be within two weeks of the date found on http://tldp.org/. If the date is not within that time period, please email feedback(at)en.tldp.org with the location of the outdated mirror. = Becoming linke to become a mirror site for the LDP, please see the Mirroring Information.

Some Mirrors Problems

Some pages may be unavailable on the mirrors. This may be a configuration problem on the mirror, or a configuration problem on the main tldp.org Web site. Please report any error to feedback(at)en.tldp.org. In the following list errors are noted when seen and not yet fixed. Forgive the inconvenience.
